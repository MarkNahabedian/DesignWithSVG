Joining Plates for T-Slot Aluminum Extrusions

This page is a tool for designing flat plates for joining T slot aluminum extrusions.

Choose the profile/size of aluminum extrusion to be joined:

The selected profile determines the unit size of an abstract grid where holes can be placed.

Choose the width and height of this grid. These are upper bounds on the size of the joining plate to be made and should be expressed as an integer multiplier of the extrusion size.
